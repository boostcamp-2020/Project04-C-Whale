할고래DO는 **알림**,  **북마크**를 지원하는 프로젝트 기반의 개인 **할 일 관리** 서비스입니다.

**웨일 확장앱 API**를 활용한 알림과 북마크 기능으로 더욱 지능적으로 할 일 관리를 해보세요 !
